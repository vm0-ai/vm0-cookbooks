# vm0-cookbooks

A collection of VM0 Agent examples.

## Installation

```bash
npm install -g @vm0/cli
```

## Quick Start

```bash
vm0 auth login
cd 101-intro
vm0 cook "echo hello world to readme.md"
```

## Authentication

Create a `.env` file in the `101-intro` directory:

```bash
# Generate token: claude setup-token
CLAUDE_CODE_OAUTH_TOKEN=<your-token>
```

## Manual Setup

For more control over artifact management:

**1. Setup artifact**
```bash
cd 101-intro
mkdir artifact && cd artifact
vm0 artifact init
vm0 artifact push
cd ..
```

**2. Compose agent**
```bash
vm0 compose vm0.yaml
```

**3. Run agent**
```bash
vm0 run intro --artifact-name artifact "echo hello world to readme.md"
```

**4. Pull results**
```bash
cd artifact
vm0 artifact pull
cat readme.md
```

## Run Without Artifact

If you don't need to retrieve files generated by the agent:

```bash
vm0 run intro "Tell me a story about cat."
```
